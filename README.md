# Kids Room Dashboard Card

A compact, structured dashboard card for a kids room:
camera, temperature, humidity, switches, occupancy, windows, and recent activity.

## Features

- Clickable switches for Kid 1 / Kid 2
- Animated occupancy indicator when motion is detected
- Color‑coded window alerts (green = closed, red = open)
- Motion + window activity timeline (based on `last_changed`)
- Clean, card‑style layout inspired by a garage dashboard design
